# Seu Caminho Financeiro

Pacote pronto para publicar no Vercel.

## Como usar
1. Baixe e descompacte este projeto.
2. Crie uma conta em https://vercel.com com seu Google.
3. Clique em **Add New Project**.
4. Escolha a opção de upload do projeto.
5. Envie esta pasta descompactada.
6. Clique em **Deploy**.

## Rodar localmente (opcional)
```bash
npm install
npm run dev
```
